Store Product Search API Docs - Below is a list of endpoints that facilitate data requests from a product serch web page: Produt Search

Request Description API Endpoint Request Method Request Headers Request Body Response Body
Create a new user https://store-item-restful-api.herokuapp.com/register POST Content-Type : Application/JSON { "username" : "name", "password" : "password" } { "message" : "User created succesfully" }
Authenticate existing user https://store-item-restful-api.herokuapp.com/auth POST Content-Type : Application/JSON { "username" : "name", "password" : "password" } { "access_token" : "token" }
Add a new item in an existing store https://store-item-restful-api.herokuapp.com/item/"string:item_name" POST Content-Type : Application/JSON, Authorization : JWT "token" { "item_price" : "price", "store_id" : "id", "store_name" : "name" } { "item_name": "name", "item_price": price, "store_id": id, "store_name": "name" }
Add a new store https://store-item-restful-api.herokuapp.com/store/"string:store_name" POST Content-Type : Application/JSON, Authorization : JWT "token" N/A { "store_name": "store_name", "store_id": id, "items": [] }
Search for all items in a store https://store-item-restful-api.herokuapp.com/store/"string:store_name" GET Content-Type : Application/JSON N/A { "store_name": "store_name", "store_id": id, "items": [] }
Search for an item in a store https://store-item-restful-api.herokuapp.com/store/"string:store_name"/"string:item_name" GET Content-Type : Application/JSON N/A [ { "item_name": "item_name", "item_price": price, "store_id": id, "store_name": "store_name" } ]
Search for an item across all stores https://store-item-restful-api.herokuapp.com/item/"string:item_name" GET Content-Type : Application/JSON N/A [ { "item_name": "item_name", "item_price": price, "store_id": id, "store_name": "store_name" } ]

Other endpoints to delete or update items are can be found here: Flask API Source Code
